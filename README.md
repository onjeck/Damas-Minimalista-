# Damas Minimalista & Estratégico 🎲✨

Uma experiência moderna e imersiva do clássico jogo de Damas, construída com tecnologias web de ponta. Este projeto combina jogabilidade tradicional com visuais arcade, efeitos 3D e modos de jogo variados.

![Damas Banner](https://picsum.photos/seed/checkers/800/400)

## 🚀 Funcionalidades Principais

### 🎮 Modos de Jogo
*   **Vs Inteligência Artificial**: Desafie a CPU em três níveis de dificuldade (Fácil, Médio, Difícil).
*   **Multijogador Local**: Jogue contra um amigo no mesmo dispositivo.
*   **Modo Online**: Simulação de partidas contra jogadores ao redor do mundo.
*   **🏆 Modo Torneio (Copa dos Campeões)**:
    *   Torneio mata-mata para 4 jogadores.
    *   Suporte a partidas "Melhor de 3" (Bo3).
    *   Visualização interativa da chave do torneio (Bracket).
*   **🎓 Tutorial Interativo**: Aprenda a jogar com lições passo a passo, desde movimentos básicos até estratégias avançadas.

### 🎨 Visual e Imersão
*   **Visualização 3D e 2D**: Alterne livremente entre uma visão tática 2D e uma experiência imersiva 3D.
*   **Controles de Câmera**: Gire e dê zoom no tabuleiro no modo 3D.
*   **Temas Personalizáveis**: Escolha entre diversos temas de tabuleiro (Clássico, Madeira, Neon, Oceano, etc.).
*   **Estilos de Peças**: Diferentes designs para as peças (Clássico, Minimalista, Tecnológico).
*   **Efeitos Visuais de Captura**:
    *   ⚡ **Raio**: Um raio atinge a peça capturada.
    *   🛸 **Abdução**: Feixe de luz alienígena.
    *   👻 **Fantasma**: A alma da peça sobe aos céus.
    *   💥 **Explosão**: Partículas voando para todos os lados.
*   **Game Juice**: Tremor de tela (Screen Shake) e partículas para aumentar o impacto das jogadas.

### ⚙️ Regras e Mecânicas
*   **Regras Configuráveis**:
    *   Captura Obrigatória.
    *   Dama Voadora (Regras Brasileiras/Internacionais).
    *   Captura para trás (Peça comum).
*   **Sistema de Histórico**: Visualize o log de todas as jogadas da partida.
*   **Desfazer Jogada**: Cometeu um erro? Volte atrás (disponível em modos offline).
*   **HUD Arcade**: Placar dinâmico, barras de vida baseadas na quantidade de peças e cronômetro.

## 🛠️ Tecnologias Utilizadas

*   **React 18**: Biblioteca principal para construção da interface.
*   **TypeScript**: Para tipagem estática e código mais seguro.
*   **Tailwind CSS**: Estilização moderna e responsiva.
*   **Vite**: Build tool rápida e eficiente.
*   **Lucide React**: Ícones vetoriais leves e bonitos.
*   **CSS Animations**: Animações fluidas para transições e efeitos.

## 🕹️ Como Jogar

1.  **Selecione um Modo**: Escolha entre jogar sozinho, com amigos ou o modo torneio.
2.  **Mova as Peças**:
    *   Clique na peça que deseja mover.
    *   Clique na casa de destino destacada.
3.  **Capture**: Pule sobre as peças adversárias para capturá-las. Lembre-se que a captura é obrigatória por padrão!
4.  **Vença**: Capture todas as peças do oponente ou deixe-o sem movimentos válidos.

## ⌨️ Controles (Modo 3D)

*   **Clique e Arraste**: Rotaciona a câmera ao redor do tabuleiro.
*   **Scroll do Mouse**: Zoom in / Zoom out.
*   **Botão Reset**: Restaura a câmera para a posição padrão.
